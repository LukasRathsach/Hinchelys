# Hinchely — Roadmap

Personal brand site for Julie og [navn]. Samler alt de laver på ét sted: virksomheder, foredrag, bestyrelsesposter, presse og content.

---

## Fase 1 — Fundament

- [ ] Modtag og gennemgå brand identity (farver, fonte, logo, billeder)
- [ ] Beslut stack (statisk HTML/CSS vs. Vite/React)
- [ ] Opsæt projektstruktur og dev-miljø
- [ ] Definer typografi og farvesystem ud fra brand identity

## Fase 2 — Sider

- [ ] Forside — fælles fortælling / deres historie
- [ ] Virksomheder — DSF, Bistro Neuf, Det Sidste Bureau
- [ ] Advisory / Bestyrelse — bestyrelsesposter
- [ ] Foredrag — emner, formater + booking-CTA
- [ ] Medie / Presse — omtale og debatindlæg
- [ ] Content hub — LinkedIn-indlæg / artikler
- [ ] Kontakt — email + signup-formular

## Fase 3 — Kvalitet

- [ ] Responsivt layout — desktop og mobil
- [ ] Visuel gennemgang og konsistenstjek
- [ ] Tilgængelighed (kontrast, semantisk HTML, focus states)
- [ ] Performance-tjek (billeder, fonthentning, ingen unødvendig JS)

## Fase 4 — Launch

- [ ] Deploy (Vercel / Netlify / andet)
- [ ] Tilslut domæne
- [ ] Smoke test på live URL
